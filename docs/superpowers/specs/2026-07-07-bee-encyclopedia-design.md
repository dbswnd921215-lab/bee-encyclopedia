날짜: 2026-07-07

## 목적

무료 백과사전 웹앱. "봄벌 키우는 방법" 같은 문장형 질문에도 정리된 항목으로 답한다.

## 형태

- 정적 웹앱(PWA), GitHub Pages 배포 (streaks-app과 동일 방식)
- 서버·비용 없음, 오프라인 검색 가능

## 3가지 접근 방식

1. **검색**: 전체 페이지 텍스트 인덱스에서 즉시 검색. 한국어 특성상 부분문자열
   매칭 + 동의어 사전(예: 봄벌=봄철관리=춘계) 방식. 결과 = 백과 항목 + 원본 페이지 스니펫.
2. **백과사전 항목**: 주요 주제 30~50개(계절관리, 병해충, 여왕벌, 분봉, 채밀,
   월동, 밀원식물 등)를 PDF 기반으로 정리. 각 항목 = 핵심 요약 + 도식(달력,
   절차도, 비교표, HTML/SVG로 새로 제작) + 출처(책·페이지) + 원본 보기 링크.
3. **원본 보기**: 페이지를 압축 WebP 이미지로 변환(전체 약 200MB 예상),
   필요한 페이지만 지연 로딩.

## 제작 단계

- **1단계**: 텍스트 추출 → 검색 인덱스 → 앱 뼈대(검색 + 원본 페이지 뷰어) → 배포
- **2단계**: 백과사전 항목을 주제 배치별로 채움 (병해충 → 계절관리 → ...), 도식 포함
- 스캔본 OCR은 필요 시 후속 단계

## 한계 (사용자 공유됨)

- 백과 항목은 단계적으로 확충. 검색은 텍스트 추출 가능한 권부터 전체 커버.
- AI 실시간 문장 생성이 아니라 미리 정리한 항목이 뜨는 방식.
